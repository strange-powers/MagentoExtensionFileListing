# Magento 1 Extension Destruction 
This little tool is for everybody who has been struggling with removing a Magento 1 Extension without the original extension Files.
## Usage
This tool is supposed to be used from the cli (command line interface).

Make sure that you moved the files in the root directory of your Magento installation.

First of all navigate into the magento-extension-destruction directoryand use the following commands to start the script:

```php run.php <name-of-the-extension>```

Where the name of the extension has to be the Magento Extension Identifier:
```Iways_PayPalPlus```

```Note: ``` This tool is in development and can currently not be used for productive usage
